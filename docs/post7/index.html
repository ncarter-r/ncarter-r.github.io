Unidad 2: Bocados Hexaédricos

Concepto

La unión de la once, merdienda de media tarde, con los solidos platonicos, poligonos reglares los cuales se caracterizan por tener todas las caras iguales, hacen de este plato una preparación en la cual se pueden encontrar distintos sabores, texturas, colores, etc., como uno esperaría de una once tradicional, pero con un factor forma que le da una presentación distinta. También se busca hacer un juego con los elementos que lo componen, pues dan una idea de un juego de encaje, en el cual cada elemento tiene un lugar hecho para cada uno, tanto en la preparación como en el plato propuesto.

Bocado Hexaédrico

El plato consiste principalmente de hexaedros (cubos) de pan o queque los cuales estan rellenos. Para lograr esto se corta con una herramienta de corte la cual debimos diseñar e imprimir en 3D. Para complementar la preparación se sirve con un octaedro de chocolate en una taza la cual tiene un espacio hecho para que el chocolate siga este juego de encaje que se busca hacer con todo el plato, además de ser transparente para que se pueda ver que es lo que esta pasando.
