Post 2.1 Tutoriales de Fusion

Tutorial 1

El primer tutorial es una introducción al programa, hicimos una suerte de molde circular. Para haber usado Inventor antes diria que las habilidades se traspasan sin problema.

Tutorial 2

Para el segundo tutorial aprendimos como hacer un molde a partir de una figura que generamos usando rieles. Lo entretenido que tiene el programa es que se puede modificar el modelo con los sketch en tiempo real, lo que te deja corregir errores de manera muy comoda, además de poder mantener un buen orden con la barra de abajo que señalaba las herramientas usadas recientemente.

Tutorial 3

Finalmente, aprendimos a hacer un molde pero esta vez más parecido a una prensadora. Hay dos partes, una base con el vaciado de la figura y la prensa en si, que tambien tiene un nevatido de la figura base y que tiene una manilla para que se pueda utilizar una vez fabricada. ![Imagen Simple](/img/Screenshot 2021-04-26 011804.png)
